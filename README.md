# Hotel Shiva Inn Website

Premium Next.js 15 website for Hotel Shiva Inn, Haveri, Karnataka.

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Image Configuration

All image URLs are centralized in `lib/images.ts`. Replace the current placeholder photography with the final Cloudflare R2 URLs when the bucket is ready.

## Included

- Full responsive landing website
- Hero slider and availability enquiry popup
- WhatsApp enquiry redirect to `+91 63606 44158`
- Rooms, dining, vegetarian dining, bar and lounge, banquets, gallery, map and footer
- Metadata, OpenGraph, Twitter card, robots, sitemap and Hotel/LocalBusiness structured data
